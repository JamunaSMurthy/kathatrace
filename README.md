# KathaTrace: Diagnosing Semantic Trajectory Collapse in Generated Visual Narratives

## Overview

KathaTrace is a generator-agnostic protocol for diagnosing semantic trajectory collapse in generated visual narratives. It includes:

- **KathaBench-25K**: A comprehensive benchmark for story visualization and storyboard generation
- **Semantic Trajectory Gap (STG)**: A metric for evaluating semantic coherence in visual narratives
- **Semantic Compass**: Tools for analyzing and visualizing semantic trajectories
